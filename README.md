# MedNote - 癫痫记录App

一个简洁易用的Android应用，用于记录癫痫发作情况。

## 功能特性

1. **简洁界面** - 清晰的输入界面，方便快速填写记录
2. **多种记录类型** - 支持记录大串、小串、强直、阵挛四种发作类型
3. **日期管理** - 可以选择日期查看和填写不同日期的记录
4. **历史记录** - 查看所有历史记录，按日期分组展示
5. **Excel导出** - 一键导出所有记录到Excel文件
6. **华为/荣耀适配** - 针对华为和荣耀手机进行优化

## 使用说明

1. 打开应用，默认显示今日日期
2. 在输入框中填写记录，格式为"时间,个数"
3. 点击"保存记录"按钮保存
4. 点击右上角的历史记录图标查看所有历史
5. 点击导出图标导出Excel文件

## 技术栈

- Kotlin
- Jetpack Compose
- Room Database
- Apache POI (Excel导出)
- Material Design 3

## 构建说明

1. 使用Android Studio打开项目
2. 等待Gradle同步完成
3. 连接Android设备或启动模拟器
4. 点击Run按钮构建并安装应用

## 权限说明

- 存储权限：用于导出Excel文件
- 网络权限：用于华为/荣耀服务（可选）

## 导出格式

导出的Excel文件包含以下列：
- 日期
- 大串
- 小串
- 强直
- 阵挛

文件保存在设备的Downloads文件夹中。
